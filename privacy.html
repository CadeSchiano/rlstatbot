RL Stat Bot – Privacy Policy

Last Updated: 12/1/2025

1. Overview

This policy explains what information RL Stat Bot collects and how it is used. The Bot collects only the minimum data needed to function.

2. Information Collected

  • User-Provided: Steam ID
  • Automatically Processed: Public Rocket League stats

The Bot does not collect emails, passwords, payment info, or sensitive data.

3. How Data Is Used

Data is used only to fetch and display requested Rocket League statistics in Discord.

4. Data Storage

Steam IDs are stored temporarily in memory and deleted when the Bot disconnects or is overwritten. No long-term database is used.

5. Data Sharing

The Bot only displays data when requested in Discord. No third-party sharing occurs.

6. Security

Reasonable safeguards are used, but no system is fully secure.

7. Children’s Privacy

RL Stat Bot is intended for users 13+ as required by Discord.

8. Changes

This policy may be updated. Continued use means acceptance of any changes.
